Description

Labyrinth Adventure merupakan sebuah permainan maze challenge berbasis 3D first-person perspective. Game ini dikembangkan penuh menggunakan framework 3js untuk membangun model 3d sekaligus interaksi yang interaktif yang ada di dalamnya.
Kami juga menerapkan beberapa algoritma khusus seperti maze generator untuk menggenerate rute maze yang ada setiap kali permainan akan dimulai.
